# 이젠기업 리뉴얼 (Ezen Corp Renewal)

본 프로젝트는 **롯데건설 웹사이트 리뉴얼 퍼블리싱 작업**입니다.
기존 사이트를 기반으로 하여 **리뉴얼**했으며,
**HTML / CSS / JavaScript**를 사용해 정적 웹사이트로 구현했습니다.
메인 비주얼 및 콘텐츠 영역에는 **Swiper 플러그인**을 적용해 사용자 경험을 강화했습니다.

---

## 프로젝트 개요

-   **프로젝트명**: 롯데건설 리뉴얼
-   **작업 유형**: 웹사이트 리뉴얼 퍼블리싱
-   **디자인**: 리뉴얼
-   **개발 방식**:HTML / CSS / JavaScript 기반 적응형 웹 퍼블리싱
-   **배포**: GitHub 저장소 업로드

---

## 사용 기술

-   **HTML5**

    -   시맨틱 마크업 기반 구조 설계

-   **CSS3**

    -   레이아웃 및 UI 스타일링
    -   공통 스타일 및 페이지별 스타일 분리

-   **JavaScript (Vanilla JS)**

    -   DOM 제어
    -   사용자 이벤트 처리
    -   UI 상태 관리

-   **Swiper**

    -   슬라이드 UI 구현

---

## 기능 정의

### 1. 공통 UI 구성

-   **Header**

    -   로고 및 주요 메뉴 구성
    -   페이지 이동을 위한 내비게이션 역할

-   **Footer**

    -   기업 정보
    -   카피라이트 영역 구성

---

### 2. 메인 비주얼 / 슬라이드 기능 (Swiper)

Swiper 플러그인을 활용하여 메인 비주얼 영역을 구성했습니다.

-   자동 슬라이드 재생 (autoplay)
-   무한 반복 슬라이드 (loop)
-   페이지네이션 표시
-   마우스/터치 드래그를 통한 슬라이드 이동
-   슬라이드별 이미지 및 텍스트 콘텐츠 출력

---

### 3. JavaScript UI 기능

-   **콘텐츠 제어**

    -   클릭 이벤트를 통한 영역 전환

-   **토글 UI**

    -   버튼 클릭 시 클래스 추가/제거 방식의 상태 제어

-   **기본 인터랙션 처리**

    -   메뉴, 버튼 등 사용자 행동에 따른 동작 구현
